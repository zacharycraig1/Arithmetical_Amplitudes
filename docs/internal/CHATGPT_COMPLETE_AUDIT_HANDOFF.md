## THEOREM 1: n=5 Discriminant Identity

### Statement

For 5-point massless scattering in 4D with generic kinematics, after gauge-fixing (σ₁, σ₂, σ₃) = (0, 1, -1):

1. The CHY equations reduce to a quadratic polynomial in σ₄
2. The discriminant **Δ = c·D** where D is the Gram determinant and c > 0 is a positive constant (varies with kinematics)
3. By the Gram-Levi-Civita identity D = -ε², we have Δ < 0
4. The splitting field is **Q(i)**

### Proof Method

1. **Polynomial reduction**: Starting from 4 CHY equations in σ₄, σ₅, compute resultant to eliminate σ₅
2. **Saturation**: Remove spurious factors σ₄, (σ₄-1), (σ₄+1) from gauge collisions
3. **Discriminant computation**: Extract B² - 4AC from remaining quadratic
4. **Gram-Levi-Civita identity**: Prove D = -ε² using det(G) = det(MηMᵀ) = -(det M)²

## THEOREM: Loop χ₈ Character

### Statement

For 4-point N=4 SYM at one loop, with the standard normalization of box coefficients:

```
[A_tree · c₁ · c_s^(2) · c_t^(2)] = [−2] ∈ K*/(K*)²
```

where K = Q(s,t) is the field of kinematic invariants.

Equivalently, for all primes p of good reduction:
```
χ_p(product) = (−2/p)
```

### Coefficient Definitions

- **A_tree**: 4-point tree amplitude (Parke-Taylor normalized)
- **c₁**: Leading singularity coefficient  
- **c_s^(2)**, **c_t^(2)**: Double-cut coefficients in s and t channels

### Proof

1. Box coefficients computed via unitarity cuts
2. Product simplifies to rational function with squarefree numerator/denominator
3. Residual squareclass is [−2]
4. By multiplicativity: (−2/p) = (−1/p)·(2/p)

### Number-Theoretic Note

The character (−2/p) corresponds to Q(√−2), which has discriminant −8. This field is contained in Q(ζ₈) = Q(i, √2) (LMFDB: 4.0.256.1).

### Convention Independence

**Lemma:** Under allowed rescalings (overall phase, basis choice), the product changes by a square in K*, so [−2] is invariant.

## SUPPORTING LEMMAS

### Lemma: Gram-Levi-Civita Identity

For null 4-vectors v₁, v₂, v₃, v₄ in Minkowski space:
```
D(v₁, v₂, v₃, v₄) = -ε(v₁, v₂, v₃, v₄)²
```

**Proof:** 
Let M = [v₁; v₂; v₃; v₄]. Then G = MηMᵀ where η = diag(1,-1,-1,-1).
det(G) = det(M)·det(η)·det(Mᵀ) = (det M)²·(-1) = -(det M)².
Since det M = ε, we have D = -ε².

### Lemma: Pythagorean Null Vector Parametrization

For integers m, n, p, q:
```
v = (m² + n² + p² + q², m² + n² - p² - q², 2(mq + np), 2(nq - mp))
```
satisfies v·v = 0 in Minkowski signature (+,-,-,-).

**Proof:** Direct computation.
